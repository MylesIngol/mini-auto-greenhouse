# mini-auto-greenhouse

## Overview

An ESP32-powered automated greenhouse that monitors plant conditions and automatically maintains a healthy growing environment.

## Goals

- Monitor temperature and humidity
- Monitor soil moisture
- Automatic watering
- Cloud dashboard
- Historical data logging
- Remote monitoring

## Hardware

- ESP32
- BME280
- SSD1306 OLED
- Soil moisture sensor
- Water pump
- Water reservoir
